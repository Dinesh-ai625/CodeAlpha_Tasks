#Language Translator | CodeAlpha Internship Project

This project is a Language Translator application developed as part of the CodeAlpha Internship Program.
It translates text between multiple languages using GoogleTranslator (deep-translator package).

---

##Features
Translate text between multiple languages  
Clean and user-friendly GUI using Tkinter  
Fast response and accuracy  

---

##Technologies Used
- Python
- Tkinter GUI
- deep-translator API

---

## How to Run
1. Install libraries using the command:  
   `pip install -r requirements.txt`
2. Run the program using:  
   `python translator.py`
