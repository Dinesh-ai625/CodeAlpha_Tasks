# Weather Prediction Chatbot

A simple desktop chatbot built with **CustomTkinter** that provides live weather updates for any city using the **Open-Meteo API**.

---

## Features

- Check current weather, temperature, and wind speed.
- Converts weather codes into human-readable conditions.
- Dark mode UI using CustomTkinter.
- Easy-to-use interface for chatting with the bot.
